# College Annual Day Management System 🎉

## Overview
The **College Annual Day Management System** is a web application designed to manage and organize all aspects of your college's annual day event. With features for event scheduling, participant registration, voting, and feedback, this system provides an easy-to-use interface for all involved.

---

## Key Features 🔑

- **Event Scheduling** 📅: Admins can create and manage a detailed schedule for all events and activities.
- **Participant Registration** 📝: Students and faculty can register for various events and performances, all in a user-friendly interface.
- **Role-based Access Control** 🔐: The system offers different levels of access: **Admin**, **Faculty**, and **Students**.
- **Real-Time Notifications** 📲: Automatic notifications for event changes, registrations, and deadlines.
- **Voting and Feedback** 🗳️: Collect votes and feedback from attendees on different performances or categories.
- **Attendance Management** ✅: Admins can monitor participant attendance for each event.
- **Responsive Design** 🌐: Built using **Bootstrap** for a fully responsive, mobile-friendly design.

---

## Technologies Used 🛠️

- **Frontend**:
  - HTML5, CSS3, JavaScript
  - Bootstrap 4 (for responsive design)
- **Backend**:
  - PHP
- **Database**:
  - MySQL
- **Authentication**:
  - Basic authentication with session management in PHP
